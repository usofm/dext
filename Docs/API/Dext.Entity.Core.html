Creates a LINQ query based on a raw SQL query. If the SQL is a stored procedure, you cannot compose over it (Where, OrderBy won't work). If the SQL is a SELECT statement, depending on the provider, you might be able to compose. Parameters are bound in declaration order (first placeholder in the SQL matches AParams[0]), e.g. WHERE id = :UserId with [TValue.From(42)] — not by names like p0.
